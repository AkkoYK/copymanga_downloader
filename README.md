# copymanga_downloader
 拷贝漫画下载器
 ## 特点
1. 采用多线程下载,可以自定义章节下载,支持失败重连
2. 纯api接口访问,本地解密,运行速度快
3. 支持下载评论并绘制为图片查看
4. 无需登录配置什么的,没有限制,打开即下
5. ~~代码工整好看,可读性高~~

 ## 食用说明
 1. 右上角下载zip,解压打开,在当前窗口打开cmd
 2. 输入 `pip install -r requirements.txt`
 3. 双击 .bat 程序
